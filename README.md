# PantryIQ - Real-Time Pantry Inventory Tracking
## 🔗 Live Demo: [CLICK HERE TO VIEW](file:///C:/Users/arshi/steganography_project/dashboard.html)


## 🎯 Problem Statement
Households waste ₹50,000/year due to food spoilage. No real-time visibility into pantry inventory.

## 💡 Solution
PantryIQ uses ESP32 + Load Cell to automatically track pantry weight in real-time. When items run low, users get alerts.

## 🛠️ Tech Stack
- **Hardware:** ESP32, HX711 Load Cell Module, 10kg Load Cell Sensor
- **Cloud:** Firebase Realtime Database
- **Frontend:** HTML + JavaScript Dashboard
- **Analytics:** Databricks (Future)

## ⚡ How It Works
1. Place items on smart shelf
2. Load cell measures weight (updates every 2 seconds)
3. ESP32 sends data to Firebase
4. Dashboard shows live inventory + alerts
5. User sees: "Rice: 450g (Low Stock!)"

## 📊 Expected Impact
- 40% reduction in food waste
- ₹20,000/year savings per household
- Scalable to 300M+ Indian homes
